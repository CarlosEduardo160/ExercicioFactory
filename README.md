## Exercício: Padrões Factory

Neste exercício, busquei praticar o uso dos padrões de criação do tipo Factory (Simple Factory, Factory Method e Abstract Factory), com o objetivo de entender melhor como separar a lógica de criação de objetos da lógica de uso. A ideia principal foi evitar o uso direto de "new" espalhado pelo código, centralizando e organizando a criação de objetos de forma mais flexível e escalável.
